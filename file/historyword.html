HISTORY 0F MICROSOFT WORD

Microsoft Pertama diterbitkan pada 1983 dengan nama  Multi-Tool Word untuk xenix, versi-versi lain kemudian dikembangkan untuk berbagai sistem operasi, misalnya dos(1983) ,apple Macintosh (1984), SCO UNIX,  OS/2, dan Microsoft Windows (1989). Setelah menjadi bagian dari Microsoft Office System 2003 dan 2007 diberi nama Microsoft Office Word. Di Office 2013, Namanya cukup dinamakan Word.
